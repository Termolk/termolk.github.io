3.12 WTF Forms - прием данных

Когда форма готова и отправлена, остается еще половина работы - данные формы нужно принять и куда-то записать.

В этом топике вы научитесь:

Проверять, пришли ли данные из форм.

Обрабатывать данные из форм.

Сохранять их в БД.

К сожалению, пока мы не можем сделать задания на Степике по этой теме, поскольку form не хочет работать в песочнице, но к каждому шагу приложен пример кода и вы сможете запустить его локально.

Полную документацию по WTForms можно найти здесь:https://wtforms.readthedocs.io/en/stable/index.html

Первый пример с формой

Итак, у нас есть форма:

class OrderForm(FlaskForm):
    name = StringField()
    phone = StringField()

И шаблон для ее вывода:

<form action="/save/" method="POST">

    <h2>Форма</h2>

    {{ form.name }}
    {{ form.phone }}

    <input type="submit">

</form>

И представление, где мы выводим эту форму:

@app.route('/form/')
def render_form():
    form = OrderForm()
    return render_template("form.html", form=form)

Пора научиться ловить момент, когда форма отправлена.

Добавим еще одно представление, где примем данные:

@app.route('/save/',methods=["GET","POST"])
def render_save():

    if request.method == 'POST':
        return "Форма отправлена"else:return "Просто зашли посмотреть"

Теперь, если мы зайдем на /save, мы увидим сообщение "Просто зашли посмотреть". Если же мы зайдем на /form и отправим форму, то на страничке /save увидим сообщение "Форма отправлена".

Большой шаг вперед для маленькой формы! Попробуйте проделать это сами!

Получение данных

Когда данные переданы, значение каждого поля можно получить через form.имяполя.data. Например, у нас есть представление, которое показывает форму:

@app.route("/form/")def render_form():# создаем форму
    form = OrderForm()

    # выводим форму
    render_template("form.html", form=form)

Добавим к нему еще одно представление, которое будет принимать форму и печатать результат.Чтобы получить результат, сделаем так:

@app.route("/save/",methods=["POST"])
	def render_save():# создаем форму
    form = OrderForm()

    # получаем данные
    name = form.name.data
    phone = form.phone.data

    # выводим данные
    return "Получены данные {} и {}".format(name,phone)

На самом деле мы не будем выводить такой текст в браузер, скорее мы сделаем так:

@app.route("/save/",methods=["POST"])
	def render_save():# создаем форму
    form = OrderForm()

    # получаем данные
    name = form.name.data
    phone = form.phone.data

    # выводим данные
    return render_template("save.html", name=name, phone=phone)

Отлично, мы уже умеем получать данные.

Но так ли нам нужны два разных представления?

Форма и получение данных в одном представлении

Допустим теперь, что мы не хотим выполнять переход после отправки формы.

Это значит, что внутри представления нам нужно будет проверять, отправлена ли форма – тогда мы обработаем данные, или еще нет – тогда мы выведем форму.

@app.route("/form/", methods=["GET","POST"])
def render_form():# создаем форму
    form = OrderForm()

    # Если данные были отправлены
		if request.method == "POST":
        # получаем данные
        name = form.name.data
        phone = form.phone.data

        # выводим данные
				return render_template("save.html", name=name, phone=phone)

    # Если данные еще НЕ были отправлены
		else:

        # выводим форму
				return render_template("form.html", form=form)

Полный листинг:

app.py

from flask import Flask, render_template, request
from flask_wtf import FlaskForm
from wtforms import StringField

app = Flask(__name__)
app.secret_key = "randomstring"
# создание формы
class OrderForm(FlaskForm):
    name = StringField("Имя")
    phone = StringField("Телефон")


@app.route("/form/", methods=["GET", "POST"])
def render_form():
		# создаем форму
    form = OrderForm()

    # Если данные были отправлены
		if request.method == "POST":

        # получаем данные
        name = form.name.data
        phone = form.phone.data

        # выводим данные
						return render_template("save.html", name=name, phone=phone)

    # Если данные еще НЕ были отправлены
		else:

        # выводим форму
				return render_template("form.html", form=form)


app.run(debug=True, port=8881)
