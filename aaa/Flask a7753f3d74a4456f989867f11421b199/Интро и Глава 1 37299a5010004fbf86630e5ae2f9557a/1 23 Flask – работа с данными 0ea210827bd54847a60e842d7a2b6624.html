1.23 Flask – работа с данными

Мы научились создавать роуты и выводить какие-то тексты в них.

Теперь мы хотим, чтобы вы научились выводить в роутах ваши данные: переменные, списки и словари в формате html.

Так данные, которые вы раньше выводили в командную строку, вы будете выводить в виде простого html, который отправится в браузер пользователя.

В этом блоке мы научимся использовать шаблоны для того, чтобы выводить эти данные.

Прежде, чем начать

В этом топике мы будем выводить html вручную в учебных целях.

В дальнейшем мы будем использовать рендеринг шаблонов с помощью Jinja.

Внутри функции

Несмотря на наличие декоратора @app.route(), функция, которая описывает роут – почти обычная функция.

В ее теле можно писать код, обращаться к объявленным переменным, вызывать другие функции.

Например, можно делать print(), чтобы вывести что-нибудь в консоль.

Функция выполняется в момент обращения к роуту.

В браузере пользователь видит то, что функция возвращает c помощью return.

Пример

Такая функция покажет в браузере текст "Привет, я функция Example":

@app.route("/example/")
def render_example():
    return "<h2>Привет, я функция Example </h2>"

Получение данных из браузера

Часть адресной строки используется для определения роута, остальные данные можно получить и использовать в функции.

Например, можно получить из адресной строки:– название раздела в каталоге;

– имя пользователя;

– филиал компании;

– название мероприятия, на которое вы покупаете билеты.
