Шаги проекта:

0. Начните работу над проектом.

– Создайте в новой папке файл app.py.

– Установите Flask.

1. Распишите роуты, выведите тексты: – главной /, (выведите здесь будет главная);– направления /departures/<departure>/ (выведите здесь будет направление);– тура /tours/<id>/ (выведите здесь будет тур).

2. Скопируйте шаблоны в папку templates:

– index.html главной;– departure.html направления;– tour.html тура.

3. Выведите нужный шаблон для каждого роута:

– index.html для главной;– departure.html для направления;– tour.html для тура.

Отлично! Теперь скопируйте код app.py в поле ниже.

(Вы также можете использовать Github для публикации своего кода, в таком случае скопируйте ссылку на репозиторий.)

Бонусная часть:

Вы уже вывели шаблоны. Засовывание данных в шаблон - дело следующей главы, но мы можем подготовить сами данные, получив их из нашего списка туров и вывести их чистым HTML без стилей. Если вы готовы – давайте сделаем это! Какие данные нам понадобятся?

Во-первых – все туры, мы выведем их на страничке с адресом /data мы назвали ее так, поскольку это будет временная страничка для вывода данных.

Во-вторых – туры по направлению, мы выведем их на страничках типа /data/departures/<названиенаправления>, например, /data/departures/<nsk>, на этой страничке будут только туры из Новосибирска

В-третьих – каждый конкретный тур, мы выведем его на страничках типа  /data/tours/<номертура>/, для каждого тура такая страничка получится своя

Итак, задание:1. Выведите на отдельной странице /data/ все туры.

Код вашей страницы должен выглядеть так:

<h1>Все туры:</h1><p>Куба: <a href="/data/tours/1/">Marina Lake Hotel & Spa 62000 4* </a></p><p>Вьетнам: <a href="/data/tours/2/">Baroque Hotel 85000 5* </a></p><p>Пакистан: <a href="/data/tours/3/">Voyager Resort 63000 3* </a></p>...

Сама страница /data/ должна выглядеть так:
