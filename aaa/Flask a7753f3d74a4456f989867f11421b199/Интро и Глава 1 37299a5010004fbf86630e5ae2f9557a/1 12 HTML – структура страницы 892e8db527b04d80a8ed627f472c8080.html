1.12 HTML – структура страницы

Программирование не такое отстраненное и механичное, как может показаться: существует множество сходств с человеческим миром. Это очень легко заметить, если вы посмотрите на структуру HTML-страниц, которая содержит такие элементы, как <head> и <body>. Написание сайта, таким образом, является созидательным процессом, и, как создатель, вы должны быть знакомы со всеми необходимыми инструментами. Научиться следует еще многому, но начать стоит с самых основ: так что, знакомьтесь — HTML-теги.

Основные теги HTML

Язык гипертекстовой разметки включает в себя набор тегов (дескрипторов), которые формируют веб-страницу. Большинство тегов являются парными: команда начинается с тега и заканчивается двойником этого тега, который содержит элемент для обозначения конца команды. Они соответственно называются открывающими и закрывающими тегами.

Среди множества HTML-тегов есть несколько, которые считаются логическими. Структура может быть разделена на три зоны, соответствующие метки, которые являются обязательными элементами страницы: <html>, <head>, и <body>.

Взгляните на код простой HTML-страницы:

<!DOCTYPE html>
<html>
	<head>
		<title>Page title</title>
	</head>
	<body>
		<h1>This is a heading</h1>
		<p>This is a paragraph</p>
	</body>
</html>

Если вы сохраните этот код в формате HTML ( .html ) и откроете его в браузере.

Это выглядит очень просто, но с помощью HTML вы можете делать более сложные вещи: настраивать текст, управлять его визуальным представлением и отображать абзацы, формы, изображения, заголовки и таблицы. Гипертекст позволяет форматировать тексты, что делает их более удобными для пользователей Интернета. Гораздо удобнее читать текст с четкой и логической разметкой, чем продираться сквозь дебри неструктурированного текста.

Давайте вернемся к приведенному выше примеру кода и рассмотрим перечисленные теги более подробно.

  • элемент <!DOCTYPE> указывает тип текущего документа — DTD (Document Type Definition). Необходимо, чтобы браузер понимал, в соответствии с каким стандартом HTML отображать веб-страницу. Если он не указан, некоторые браузеры могут отображать непонятные символы вместо текста. Как видите, это один из тегов, который не является парным;
  • тэг <html> указывает, что это документ HTML;
  • тэг <head> предназначен для хранения элементов, которые помогают браузеру и поисковым системам работать с данными;
  • тэг <title> содержит заголовок документа HTML. Хотя этот тег не является обязательным, он все еще присутствует почти на всех веб-страницах в Интернете;
  • тэг<body> определяет область содержимого страницы. Он обрамляет содержимое, отображаемое в главном окне браузера;
  • <h1> содержит заголовок страницы внутри тела и <p> — абзац. Эти теги отвечают за размер шрифта, его выделение, а так же используются для создания иерархической структуры текста внутри страницы.
